7 Aliases
Contents
Within scripts all directory, deck, and note type names are always referenced via aliases. When a user runs a script, eg. by installing an addon built using Anki Script, or by adding lesson content , a dialog is displayed to the user permitting them to override a default alias value specified in the script.
This provides a mechanism for users to names decks and cards as they wish, whilst still being able to use a domain specific/course addon: ...
